# Mininet Common

This repository defines the common libraries used by the various Mininet environments.

## Dependencies

1. sysctl: https://linux.die.net/man/8/sysctl
2. ethtool: https://linux.die.net/man/8/ethtool
3. simple_switch_grpc: https://github.com/p4lang/behavioral-model/blob/main/targets/simple_switch_grpc/README.md
4. frr: https://frrouting.org/
5. bird: https://bird.network.cz/
6. iperf3: https://iperf.fr/
